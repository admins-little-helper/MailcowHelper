# Remove-MHQueue

## SYNOPSIS
Delete the current mail queue.
This will delete all mails in the queue.

## SYNTAX

```
Remove-MHQueue [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Delete the current mail queue.
This will delete all mails in the queue.
